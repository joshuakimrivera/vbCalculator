# Windows 10 Standard Calculator
### A simple knockoff of the windows 10 standard calculator using Visual Basic



#### Functionalities
* Addition, Subraction, Multiplication & Division Operations
* CE (Clear Entry) Button
* C (Clear) Button
* Negate Button
* Decimal Button
* Numerical Buttons
